#### Para Correr el proyecto haz lo siguiente:

> Descárgalo

> Descomprimelo, y luego dentro de la carpeta proyecto-tienda haz lo siguiente:

> npm install

#### Luego de que haya instalado haz:

> npm run start:dev

y listo, ya tendrías que tener corriendo el proyecto
